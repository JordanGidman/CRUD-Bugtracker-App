TEAMCODE

The teamcode allows for multiple users to work on and share the same projects and bugs.

Simply enter any word or number or combination of both and click create. This will create a new team with that code. Now any other users can enter that same code and click join and they will be able to create read update and delete bugs and projects for the entire team. If you would like some sample data please enter "hireme" in the teamcode and press join.

If you refresh the page you will need to enter the teamcode again or navigate to the team tasks section from within a project.
